02. Fondamentali

Cosa imparare nelle prime settimane

Variabili e tipi

Stringhe, numeri, booleani, liste, dizionari. Capire i tipi ti evita tanti bug.

Condizioni

if/elif/else per decidere comportamenti in base ai dati.

Cicli

for e while per automatizzare operazioni ripetitive.

Funzioni

Rendi il codice riutilizzabile e più leggibile dividendo in blocchi logici.

Esempio base completo

main.py 
def calcola_media(voti):
    if not voti:
        return 0
    return sum(voti) / len(voti)

def esito(media):
    if media >= 6:
        return "Promosso"
    return "Da recuperare"

lista_voti = [7, 6, 8, 5]
media = calcola_media(lista_voti)
print(f"Media: {media:.2f}")
print(f"Esito: {esito(media)}")

09. FAQ

Domande frequenti di chi inizia

Devo imparare tutto prima di fare progetti?

No. Parti subito con mini progetti: impari più velocemente facendo.

Se non capisco gli errori?

Leggi la riga esatta indicata, isola il problema in un file piccolo e prova una modifica alla volta.

Quanto tempo serve per diventare autonomo?

Con costanza quotidiana, già in 4-8 settimane puoi creare script utili da solo.
